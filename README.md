## Vertex AI - GEMINI - API

### Prerequisites
> * JDK 21
> * Apache Maven
> * Git

### How to start local version
```bash
mvn clean spring-boot:run -Dspring-boot.run.arguments=--GOOGLE_APPLICATION_CREDENTIALS=$PATH_TO_YOUR_KEY
```
